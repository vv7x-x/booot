# WhatsApp Downloader Bot 🚀

بوت واتساب احترافي لتحميل الفيديو والصوت من مختلف المنصات (YouTube, TikTok, Instagram, Facebook, etc.) باستخدام Node.js ومكتبة Baileys و yt-dlp.

## المميزات
- ✅ دعم أكثر من 1000 موقع فيديو.
- ✅ إرسال معلومات الفيديو (العنوان، المدة، الصورة المصغرة) قبل التحميل.
- ✅ اختيار صيغة التحميل (MP3 أو MP4).
- ✅ اختيار جودة الفيديو (360p, 720p, 1080p).
- ✅ حذف الملفات تلقائياً بعد الإرسال لتوفير المساحة.
- ✅ نظام حالات (State System) للتعامل مع طلبات المستخدمين المتزامنة.
- ✅ مستقر وسريع مع معالجة الأخطاء.

## المتطلبات
- [Node.js](https://nodejs.org/) (إصدار 16 أو أحدث).
- [FFmpeg](https://ffmpeg.org/download.html) (ضروري لدمج الصوت والفيديو وتحويل الصيغ).

## التثبيت والتشغيل

1. **تثبيت التبعيات:**
   ```bash
   npm install
   ```

2. **التأكد من وجود FFmpeg:**
   يجب أن يكون FFmpeg مثبتاً ومضافاً إلى الـ PATH في جهازك.

3. **تشغيل البوت:**
   ```bash
   node index.js
   ```

4. **ربط الحساب:**
   سيظهر رمز QR Code في التيرمنال، قم بمسحه باستخدام تطبيق واتساب (الأجهزة المرتبطة).

## كيفية الاستخدام
- أرسل أي رابط فيديو للبوت.
- سيقوم البوت بإرسال تفاصيل الفيديو وسؤالك عن الصيغة (MP3/MP4).
- إذا اخترت MP4، سيطلب منك اختيار الجودة.
- سيقوم البوت بتحميل الملف وإرساله لك، ثم حذفه من السيرفر.

## ملاحظات برمجية
- تم استخدام `yt-dlp-exec` لضمان الحصول على أحدث التحديثات لتجاوز حظر المنصات.
- تم تنظيم الكود في مجلد `lib` لسهولة التطوير والصيانة.
- يتم حفظ جلسة تسجيل الدخول في مجلد `auth_info`.
